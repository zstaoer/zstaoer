    # MybatisPlus处理数据库json类型字段

    1. Student数据表有以下字段

      字段名 字段类型 示例
      id bigint 1697949587094011906
      username varchar zhangsan
      password varchar 123456
      status int 0:正常 1：冻结
      info json {"hobby": "轮滑", "gender": "男", "height": 165, "weight": 98}

    2. 接下来在Java中建立如下实体，在对应的Info实体上面加上@TableField(typeHandler = JacksonTypeHandler.class)

      @Data
//autoResultMap = true 开启实体映射，类似xml中的ResultMap标签
@TableName(value = "student", autoResultMap = true)
public class Student {
    private Long id;
    private String username;
    private String password;
    private Integer status;
    //配置json解析器
    @TableField(typeHandler = JacksonTypeHandler.class)
    private StudentInfo info;
}

      # MybatisPlus处理枚举类型

      表中有一个status字段记录着学生状态，平时我们需要使用判断去赋值，如

      stu.getStatus == 0 ? stu.setDesc("正常"): stu.setDesc("冻结")
      1

      这样不仅需要额外扩展一个desc字段，还需要去判断，使用mp的方式去处理就会优雅很多

      首先我们定义一个枚举对应status字段

      @Getter
public enum StatusEnum {
    NORMAL(0, "正常"),
    FREEZE(1, "冻结");
    //需要加上这个注解,插入的时候会自动获取value
    @EnumValue
    private final int value;
    //需要加上这个注解,返回的时候会返回desc
    @JsonValue
    private final String desc;

    StatusEnum(Integer value, String desc) {
        this.value = value;
        this.desc = desc;
    }
}

        @PostMapping("saveStudent")
     public void saveStudent() {
            Student stu = new Student();
            stu.setUsername("赵丽颖");
            stu.setPassword("123456");
            //stu.setStatus(0);
         	/**
              * 以前使用Integer类型时需要获取值，如
              * stu.setStatus(StatusEnum.NORMAL.getValue());
              * 现在直接设置枚举，在枚举中的@EnumValue注解会自动获取到value
			 */
             stu.setStatus(StatusEnum.NORMAL);
            stu.setInfo(StudentInfo.of(165, 98, "羽毛球", "女"));
            Db.save(stu);
        }
//在查询的时候@JsonValue注解会自动转value为desc返给前端 如果value为0，将会返回"正常"给前端，这样不用我们手动去判断赋值，也不需要额外扩展desc字段
